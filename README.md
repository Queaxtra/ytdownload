# YTDownload
##### It is a website created using the yt1s.io site's API and allows you to download the Youtube music you want in seconds.
## Setup 

1) Download source codes.
```bash
https://github.com/Queaxtra/ytdownload
```

2) Install.

```bash
  npm install
```

3) Run the site.

```bash
  npm dev
```
